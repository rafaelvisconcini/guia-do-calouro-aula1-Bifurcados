# Guia de Sobrevivência do Calouro

Catálogo colaborativo da turma: o que a gente queria saber no primeiro dia de faculdade.
Cada dica é um "verbete" (um arquivo em `/verbetes`). Juntos, eles formam este guia.

## Como este repositório funciona

Este é o repositório-base, publicado pelo professor. Cada grupo trabalha em um **fork** dele — veja
o passo a passo em [`CONTRIBUTING.md`](CONTRIBUTING.md). Você escreve apenas Markdown.

## Comece por aqui

- [Índice completo dos verbetes](verbetes/INDEX.md) — tudo organizado por categoria.
- [Como contribuir](CONTRIBUTING.md) — o passo a passo para adicionar a sua dica.
- [Modelo de verbete](verbetes/MODELO.md) — copie e preencha.
- [Como registrar evidências](evidencias/README.md) — prints e anotações de cada aluno.

## Categorias

| Categoria | O que entra aqui |
| --- | --- |
| Alimentação | cantinas, onde comer barato |
| Transporte | ônibus, caronas, ir e voltar dos campus |
| Estudos | laboratórios, salas, biblioteca, wi-fi e tomada |
| Burocracia | matrícula, ajuste, atestados, secretaria |
| Moradia | repúblicas, aluguel, morar perto |
| Lazer | bares, eventos, atléticas |
| Sobrevivência | dicas gerais que salvam o semestre |
